# -Real-Time-Temperature-Humidity-Monitoring-AWS
Sistem monitoring IoT berbasis AWS untuk memantau data suhu dan  kelembaban  secara real-time. Data sensor dari simulator Wokwi dikirim via MQTT  ke AWS IoT  Core, diproses oleh Lambda, disimpan di RDS &amp; S3, dan ditampilkan di  dashboard  web yang berjalan di EC2.
